# Chat 사용 방법

## 모델 선택

Chat 화면 아래쪽의 **Select model** 버튼을 눌러 사용할 모델을 고릅니다.

- Provider별로 등록된 모델 목록이 표시됩니다.
- 선택한 모델은 `provider / model` 형식으로 보입니다.
- 모델을 선택하지 않으면 질문을 보낼 수 없습니다.

## 연결 상태

입력창 아래에는 연결 상태가 표시됩니다.

- `Connected`
  - 서버와 정상 연결된 상태
- `Disconnected`
  - 연결이 끊긴 상태

연결이 끊겼을 때는 **Reconnect** 버튼으로 다시 연결을 시도할 수 있습니다.

## 질문 보내기

1. 모델을 선택합니다.
2. 입력창에 질문을 씁니다.
3. 전송 버튼을 누릅니다.

모델을 선택한 뒤 빈 화면에서는 추천 질문 칩이 보일 수 있습니다.  
이 칩을 누르면 예시 질문이 입력창에 채워집니다.

예:

- 어떤 테이블이 있어?
- 태그 목록 보여줘
- 분석 대시보드 만들어줘
- 분석 리포트 작성해줘

> 스크린샷 위치: `llm-chat-suggestions.png`
>
> 권장 장면: 빈 대화 화면에서 추천 질문 칩이 보이는 화면

## 응답 생성 중 동작

응답 생성 중에는 전송 버튼 대신 **Stop** 버튼이 표시됩니다.

- 응답을 중단하고 싶으면 Stop 버튼을 누릅니다.
- 생성이 끝나면 일반 전송 버튼으로 돌아옵니다.

## 대화 지우기

대화가 쌓인 뒤에는 화면 오른쪽 위의 **Clear session** 버튼으로 현재 세션을 비울 수 있습니다.

이 동작은 현재 대화 세션을 정리하는 용도입니다.

## Settings로 돌아가기

입력 영역 오른쪽 아래의 Settings 버튼을 누르면 설정 화면으로 이동합니다.

다음 경우에 주로 사용합니다.

- 다른 Provider API Key를 추가할 때
- Model을 새로 등록할 때
- Machbase 연결 정보를 바꿀 때

## 사용 팁

- 처음에는 간단한 조회 질문으로 시작합니다.
- 대시보드나 리포트 요청은 테이블 이름을 명확히 포함하는 편이 좋습니다.
- 모델이 여러 개라면 빠른 질의용과 정밀 분석용을 나누어 등록하면 편리합니다.
